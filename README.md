# Hantavirus Analysis Dashboard

A Power BI dashboard project analyzing global hantavirus outbreak data using interactive visualizations and maps.

## 📊 Features
- Total cases, deaths, and CFR analysis
- Country-wise outbreak analysis
- Transmission type visualization
- Interactive map visualization
- Year-wise trend analysis
- Pie and bar chart insights

## 🛠 Tools Used
- Power BI
- CSV Dataset
- Data Visualization Techniques

## 📁 Project Files
- `Hantavirus_Analysis.pbix`
- `dataset.csv`
- `dashboard.png`

## 🚀 How to Run
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Refresh dataset if needed

## 📌 Project Objective
To analyze hantavirus outbreak patterns and visualize important health insights through an interactive dashboard.
